## 練習1 改進說明

原版 [練習1.cpp](練習1.cpp) 用固定大小的全域陣列 `int ans[1000]` 儲存閏年結果，
若輸入區間內的閏年數量超過 1000，寫入 `ans[cnt++]` 會超出陣列範圍，造成未定義行為。

[練習1-adviced.cpp](練習1-adviced.cpp) 改用 `std::vector<int>` 搭配 `push_back`，
陣列大小隨資料動態成長，不需要事先宣告或猜測容量上限，也就不會有越界風險。
