# C++ Practice

C++ 練習題目集。

## 題目列表

| 檔案 | 題目 |
|---|---|
| [練習1.cpp](練習1.cpp) | 輸入圓的半徑 r，輸出其直徑、圓周長和面積 |
| [練習2.cpp](練習2.cpp) | 給定三條線段的長度，判斷能否構成三角形 |
| [練習3.cpp](練習3.cpp) | 給定一個整數，判斷是奇數還是偶數 |
| [練習4.cpp](練習4.cpp) | 給定一個年份，判斷是否為閏年 |
| [練習5.cpp](練習5.cpp) | 輸入體重和身高，根據 BMI 指數判斷體型 |
| [練習6.cpp](練習6.cpp) | 給定一個整數，判斷它能否被 3、5、7 整除 |
| [練習11.cpp](練習11.cpp) | 給定一個含有 n 個整數的序列，作去重操作 |

## 編譯與執行

```bash
g++ 練習1.cpp -o 練習1
./練習1
```
